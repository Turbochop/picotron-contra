# Contra

A Lua/PICO-style Contra prototype by turbochop, with graphics work by reecegames.

Current working baseline:

- `level_type = "side scrolling"`
- `scrolling = "horizontal"`
- `map_helper(0, 80, 220)`

Recent experimental work includes camera logic, layered map caching, spawn streaming, and early support for vertical/top-down sections.
